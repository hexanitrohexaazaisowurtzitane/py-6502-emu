# 8bit MOS 6502 CPU Emulator
An 8bit MOS6502 CPU emulator made in python.
This project was heavily inspired py the tiny8 project, make sure to check it out aswell!
<img width="1131" height="767" alt="image" src="https://github.com/user-attachments/assets/91bddd09-3f25-45d5-a0b8-28ec98a33d9a" />

The program emulates the classic MOS 6502 8bit CPU, found in early systems like the NES, Apple II, and the Commodore 64.
Contains a built-in assembler and a text ui/debugger using curses.

Watch the showcase video:
https://www.youtube.com/watch?v=PaHTnMpoFQI

Usage: python main.py <filename.asm>
